    Keyboard Shortcuts

    Shortcut Action
    ctrl + [panel num] Show nth panel
    ctrl + ยง
    (or `)    		 Hide focused panel
    ctrl + enter Re-render JavaScript.
    If console visible: run JS in console
    ctrl + \ Auto hide navigation bar
    ctrl + / Toggle comment on single line
    esc Code complete (JavaScript only) and close open overlays - like this
    ctrl + s Save current Bin
    ctrl + shift + s Clone current Bin
    cmd + [ (or tab) Indents selected lines
    cmd + ] (or shift + tab) Unindents selected lines

    JS Bin also supports Emmet/Zen Coding shortcuts (except tab key)

    JS Bin URLs

    URL Action
    / Show the full rendered output.
    This content will update in real time as it's updated from the /edit url.
    /edit Edit the current bin
    /watch Follow a Code Casting session
    /embed Create an embeddable version of the bin
    /{{#home}}{{home}}{{/home}}{{^home}}[username]{{/home}}/last View the last edited bin for this user
    /{{#home}}{{home}}{{/home}}{{^home}}[username]{{/home}}/last/edit Edit the last edited bin for this user
    /{{#home}}{{home}}{{/home}}{{^home}}[username]{{/home}}/last/watch Follow the Code Casting session for the latest bin for this user
    /quiet Remove analytics and edit button from rendered output
    .js Load only the JavaScript for a bin
    .css Load only the CSS for a bin

    Except for username prefixed urls, the url may start with http://jsbin.com/abc and the url fragments can be added to the url to view it differently.
