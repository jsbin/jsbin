# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.x     | :white_check_mark: |
| 4.x.x   |:warning_mark:      |

## Reporting a Vulnerability

Please report vulnerabilities through the issues page. Bring up an issue and we should respond to you soon.
